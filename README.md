# CV React App

A modern, multipage React-based CV application that renders portfolio content from Markdown with a sleek, responsive design.

## Features

- Built with React using CDN for easy deployment
- Multipage structure: Home, Skills, Experience, Projects, Education
- Renders Markdown content as HTML
- Modern design with gradients, animations, and hover effects
- Responsive design optimized for mobile and desktop
- Font Awesome icons for visual enhancement
- Smooth scrolling and scroll-based animations
- Suitable for static hosting like GitHub Pages

## Pages

- **index.html**: Home page with professional summary
- **skills.html**: Core skills and certifications
- **experience.html**: Professional experience
- **projects.html**: Project portfolio with card layouts
- **education.html**: Education background and referees

## Deployment to GitHub Pages

1. Create a new repository on GitHub (or use an existing one).
2. Upload all files (`index.html`, `skills.html`, `experience.html`, `projects.html`, `education.html`, `style.css`, `README.md`) to the repository.
3. Go to the repository settings.
4. Scroll down to "Pages" section.
5. Under "Source", select "Deploy from a branch".
6. Choose the main branch and set the folder to `/` (root).
7. Save the settings.
8. GitHub will provide a URL where your CV is hosted (e.g., `https://username.github.io/repository-name/`).

## Local Development

Open `index.html` in a web browser to start viewing the CV. Navigate between pages using the header menu or page navigation buttons.

## Technologies Used

- React (via CDN)
- Marked.js for Markdown parsing
- Babel for JSX transformation
- Font Awesome for icons
- CSS3 for modern styling and animations
